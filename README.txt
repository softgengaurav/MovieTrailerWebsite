Download the GitHub repository from the following link:
https://github.com/softgengaurav/MovieTrailerWebsite.git


Follow the given steps to run the program:

1. You need to have Python 3.x.x installed on your System.

2. Python should also be added to your environment variables(in case of Windows)

3. Open the terminal(on Mac) or Command Prompt(on Windows) and get into the folder that you have downloaded from the GitHub repository.

4. Type-> python entertainment_center.py

5. This opens up your default Internet Browser and shows you the HTML page consisting of posters of various movies.

6. Next, you can click on any of the poster to view that movie’s trailer.
